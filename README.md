# AI in Plain English

Most AI writing either oversimplifies or overwhelms readers. This site finds the middle.

One visual post a week. Transformers. Reasoning models. Agents. Evals.
Written for curious people who want to understand AI without enrolling in a degree.

## What you will find here

12 posts covering everything from how transformers work to building agentic systems.
Each post is visual, specific, and short enough to read in one sitting.

## Built with

- [Astro](https://astro.build) for the site
- [D3.js](https://d3js.org) for visualizations
- GitHub Pages for hosting

## Run it locally

```bash
npm install
npm run dev
```

The site runs at `http://localhost:4321`.
